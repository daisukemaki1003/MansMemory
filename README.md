### Stack
- Flutter 2.10.5
- Flutter riverpod 1.0.3



## Setup project

プロジェクトをダウンロード
```bash
git clone https://github.com/salvadordeveloper/flutter-crypto-app
```

フラッターの依存関係を取得
```bash
flutter pub get
```

アプリを実行
```bash
flutter run
```
