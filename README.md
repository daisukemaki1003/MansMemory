# 人を記録するアプリ

### ターゲット
- 夜職の女性

### 目的
- 出会った人との思い出を記録する。

### 機能要件
- 記録
  - パーソナルデータ
    - 基本情報
      - 名前
      - ふりがな
      - 年齢
      - 生年月日
    - 趣味、生活
      - 趣味
      - 居住地
      - 休日
    - 学歴、職種
      - 学歴
      - 職種
      - 年収
  - 日記
- 表示
  - 作成日順

### システム構成
- フロントエンド
  - flutter
- バックエンド
  - firebase
